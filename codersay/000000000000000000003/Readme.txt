学习了PHP MVC框架ThinkPHP后，自己一直在想写点东西，于是这个博客系统自热而然就诞生了。

该博客的开发环境是：Apache + PHP + Mysql + ThinkPHP + win2003=kenblog  v1.0

就暂且取名为Kenblog 吧!

因为是基于ThinkPHP的，因此网站前台完全伪静态，同时支持数据缓存，并且各个模块独立调用，便于灵活调用。

源代码包括：数据库UML设计图，网站完整的源代码，完整的数据库文件(使用时直接导入)。

本博客的系统的特点包括：



1。网站单一入口index.php，有效提高网站的安全性。

2。完全使用面向对象开发，代码清晰明了，易于修改和二次开发。

2。由于使用了模板技术，所以全站伪静态，保证了友好的SEO效果。

3。缓存Cache技术，可以减轻服务器负担，提高网站负载能力。

4。前后台模块分离，这样可以更清晰的找到各个模块直接的调用方式，便于管理。

5。简介友好的后台管理，使用经典的后台框架，非常便于功能扩展。
